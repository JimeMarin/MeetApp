
# 🏢 MeetApp — Meeting Room Scheduler

A web application built with **React** and **Firebase** that streamlines the process of booking meeting rooms within an organization — with automated email notifications and conflict prevention.

## ✨ Features

### 📅 Room Booking
- Book available meeting rooms by date and time
- Automatic **conflict detection** — prevents double bookings
- View room availability in real time

### 📧 Email Notifications
- **Confirmation email** sent to the organizer upon booking
- **Invitation emails** sent to all meeting attendees
- Notifications on booking updates or cancellations

### 🔐 Authentication
- Secure login system with **Firebase Authentication**
- Password change functionality
- Role-based access: **Admin** and **User**

### 🛠️ Admin Panel
- Create and manage meeting rooms
- Edit room details and availability
- **Block rooms** when unavailable (maintenance, special events, etc.)
- Create and manage user profiles

## 🧰 Tech Stack

| Layer | Technology |
|---|---|
| Frontend | React |
| Authentication | Firebase Auth |
| Database | Firebase Firestore |
| Email Service | EmailJS |
